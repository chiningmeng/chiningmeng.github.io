运算符重载

重载输出运算符<<

  1. 输出运算符的第一个形参是一个非常量ostream对象的引用。
    非常量的原因：向流写入内容会改变其状态；
    该形参是引用的原因：我们无法直接复制一个ostream对象。
  2. 第二个形参一般是一个常引用（我们想打印出来的类类型）
    该形参是引用的原因：希望避免复制实参；
    该形参是常量的原因：打印对象不会改变对象的内容
    为与其他输出运算符保持一致，operator<<一般要返回它的ostream形参
ostream& operator<<(ostream& os, const Sales_item) {
	os << item.isbn() << " " << item.units_sold << " "
		<< item.revenue << " " << item.avg_price;
	reutrn os;
}

注意输出运算符要尽量减少格式化操作

让用户有权控制输出细节

否则，左侧的运算对象将是我们类的一个对象
